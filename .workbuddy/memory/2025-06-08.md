# 2025-06-08

## VSCode Extension: xml2udl 预览插件

### 完成内容
- 基于 demo 目录下的 xml2udl.html + xml2Udl.js 的解析逻辑，开发了一个 VSCode 扩展插件
- 插件位于 `vscode-extension/` 目录下

### 架构设计
1. **xml2UdlEngine.ts** — 核心解析引擎，从浏览器版 xml2Udl.js 完整移植到 TypeScript
   - 使用 @xmldom/xmldom 替代浏览器 DOMParser
   - 保留完整的 _META 元数据定义（Class/Parameter/Property/Index/Method/Query 等）
   - 保留 FormalSpec 状态机解析器
   - 导出 parseXmlFile、generateUdlFromXmlElement 等函数

2. **treeDataProvider.ts** — TreeView 数据提供者
   - 按类型（Class/Property/Method 等）分组显示解析后的 XML 项
   - 支持展开/折叠，显示各类型 item 数量
   - 点击叶子节点触发 openUdlDocument

3. **udlDocumentProvider.ts** — 虚拟文档提供者
   - 使用 `xml2udl-udl://` URI scheme 生成临时文档
   - 文档利用 VSCode 原生语法高亮（cos/javascript/html/xml）
   - 在右侧编辑器列打开，不产生磁盘文件

4. **extension.ts** — 入口点
   - 注册右键菜单命令（.xml 文件，Explorer 和编辑器上下文菜单均可触发）
   - 注册 TreeView 到 Explorer 侧边栏（视图 ID: xml2udl.treeView）
   - 注册 xml2udl.openItem 命令（树节点点击）

### 编译状态
- TypeScript 编译通过，零错误
- 依赖：@xmldom/xmldom, @types/vscode, @types/node, typescript

### 使用方式
1. 右键 .xml 文件 → 选择 "xml2udl预览"
2. 侧边栏显示按类型分组的项目列表
3. 点击具体项 → 右侧编辑器显示生成的 UDL 内容（带语法高亮）
